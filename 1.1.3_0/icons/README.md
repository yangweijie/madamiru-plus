# Icônes de l'extension

Ce dossier doit contenir les icônes de l'extension aux dimensions suivantes :
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Création des icônes

Vous pouvez créer vos propres icônes ou utiliser un générateur en ligne.

### Suggestions de design :
- Logo représentant l'upscaling (flèches vers le haut, pixels, résolution HD/4K)
- Couleurs : violet/bleu (comme dans le gradient de l'interface)
- Style moderne et minimaliste

### Outils recommandés :
- Figma
- Canva
- GIMP
- Photoshop
- Générateurs d'icônes en ligne
